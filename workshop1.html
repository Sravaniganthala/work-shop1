2.what are the semantic elements in html?.

Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. They help to create a well-structured and meaningful web page, improving accessibility, SEO, and maintainability. Here are some common semantic elements and their purposes: 

header: The header represents introduction content or a set of navigation links,typically found at the top of a page or section.
 
nav :Defines a navigation section, usually connecting links to other pages tor  sections of website.
    
main : Represents thedominant   content of the body, eexcluding headers, footers,and sidebars. There should be only one main element per document.
    
section : Defines a section of content, typically with a heading. It groups related content together.
    
article : Represents a self-contained piece of content that could be distributed independently, such as a blog post, news article, or forum post.
    
aside : Contains content that is tangentially related to the content around it, such as sidebars, pull quotes, or advertisements.
    
footer : Contains footer content for a section or the entire page, such as contact information, copyright notices, or links to related documents.
    
figure : Used to encapsulate media (like images, illustrations, or videos) and its caption, which is specified with the figcaption element.
    
figcaption: Provides a caption or description for the content inside a figure.
    
time: Represents a specific period in time, such as a date or time, and allows for better machine readability.
    
address: Provides contact information for the author or owner of a document or section.
    
details: Allows the user to view or hide additional information, typically shown or hidden with a toggle.
    
summary: Provides a summary or heading for the content inside a details element.
    
mark: Highlights parts of text that have relevance or importance, often used to emphasize a keyword or phrase.
    
label: Defines a label for an input element, improving form accessibility by associating labels with form controls.
    
Using semantic elements helps both browsers and assistive technologies understand the structure and content of a web page better, leading to improved user experience and accessibility.

5.what are list tags and how many are they? 
HTML lists allow web developers to group a set of related items in lists.

        Unordered HTML List:
        An unordered list starts with the ul tag. Each list item starts with the li tag.
        
        The list items will be marked with bullets (small black circles) by default.
        
        Ordered HTML List:
        An ordered list starts with the ol tag. Each list item starts with the li tag.
        
        The list items will be marked with numbers by default.
        
        Description list:
        A description list is a list of terms, with a description of each term.
